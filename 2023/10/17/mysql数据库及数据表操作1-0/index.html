mysql数据库及数据表操作1.0

mysql数据库及数据表操作

1.登入

win+r => cmd => mysql -u root -r => 回车

2.查询数据库及数据表

a.查询所有数据库

show database;

b.查询当前数据库的所有表

show tables;

c.查看一个表的结构

desc name(表名)

3.创建数据库和数据表

a.创建数据库

create database name（库名）

b.创建数据表

create table name (

列名1 数据类型1,

….

列名n 数据类型n,

);

mysql> SHOW CREATE TABLE students;
+———-+——————————————————-+
| students | CREATE TABLE students ( |
| | id bigint(20) NOT NULL AUTO_INCREMENT, |
| | class_id bigint(20) NOT NULL, |
| | name varchar(100) NOT NULL, |
| | gender varchar(1) NOT NULL, |
| | score int(11) NOT NULL, |
| | PRIMARY KEY (id) |
| | ) ENGINE=InnoDB AUTO_INCREMENT=1 DEFAULT CHARSET=utf8 |
+———-+——————————————————-+

上图表明了怎么写表；

4.使用数据库

use name(库名);

5.修改数据表(代码上，作用下 )

提醒(无特殊说明时，name即是表名)

alter table name rename to newname;

(改表名)

alter table name character set 字符集名称;

(修改表的字符集)

alter table name add name(列名) 数据类型;

(添加一列)

alter table name change name(列名) newname(新列别) 新数据类型;

(修改列名称 类型1)

alter table name modify name(列名) 新数据类型;

(修改列名称 类型2)

alter table name drop name(列名)

(删除列)

6.删除数据库及数据表

删除数据库

drop database if exists name(库名);

删除数据表

drop table if exists name(表名);
