mysql数据库及数据表操作

1.登入

win+r => cmd => mysql -u root -r => 回车

2.查询数据库及数据表

a.查询所有数据库

show database;

b.查询当前数据库的所有表

show tables;

c.查看一个表的结构

desc name(表名)

3.创建数据库和数据表

a.创建数据库

create database name（库名）

b.创建数据表

create table name (

列名1 数据类型1,

….

列名n 数据类型n,

);
